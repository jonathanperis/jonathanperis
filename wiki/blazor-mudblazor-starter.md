# Blazor MudBlazor Starter

> Blazor Server starter template with MudBlazor Material Design components

**Repository**: [github.com/jonathanperis/blazor-mudblazor-starterL(https://github.com/jonathanperis/blazor-mudblazor-starter.md) | **Stars**: 1 | **Language**: C#

## Overview

A ready-to-use Blazor Server template pre-configured with MudBlazor 8.3, the Material Design component library. Designed to eliminate the boilerplate of starting a new Blazor project with a polished UI framework.

## Features

- **MudBlazor 8.3** -- Material Design components out of the box
- **Demo pages** -- Home, Counter, Weather forecast with CRUD dialogs
- **.NET 9** with AOT compilation and trimming enabled
- **Multi-architecture Docker** -- builds for both AMD64 and ARM64
- **CI/CD pipeline** -- GitHub Actions deploys to Docker Hub and Azure Web App
- **Live demo** hosted on Azure (Brazil South region)

## Infrastructure

| Component | Details |
|-----------|---------|
| Runtime | .NET 9 (Blazor Server) |
| UI Framework | MudBlazor 8.3 |
| Container | Multi-arch Docker (AMD64 + ARM64) |
| CI/CD | GitHub Actions |
| Registry | Docker Hub |
| Hosting | Azure Web App (Brazil South) |

## Use Case

This template is useful when you need to:
- Spin up a new Blazor Server app with Material Design quickly
- Have Docker and Azure deployment ready from the start
- Start with working CRUD patterns (dialogs, forms, tables)
